READ ME:

To run:
CMake the project 
1. In terminal navigate to project root.
2. Type the following: 
	cmake -G"Eclipse CDT4 - Unix Makefiles"
cmake -G"Eclipse CDT4 - Unix Makefiles" -DCMAKE_ECLIPSE_VERSION=4.4 -DCMAKE_OSX_ARCHITECTURES=x86_64

   If the compiler is not Eclipse adjust the generator variable.
